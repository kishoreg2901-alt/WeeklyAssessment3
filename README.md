# SQL Foundations Weekly Assessment

This repository contains my completed solutions for the SQL Foundations Weekly Assessment, developed using MySQL Workbench. The assessment is based on a fictional Flipcart e-commerce product catalog and is designed to strengthen fundamental SQL concepts through both theoretical and hands-on exercises.

The project begins with setting up the database using the provided SQL script, followed by solving a series of theory questions, output prediction tasks, and practical SQL problems. The queries demonstrate essential SQL operations such as data retrieval, filtering, sorting, pattern matching, aggregate functions, grouping, HAVING clauses, CASE expressions, NULL handling, and analytical reporting.

Throughout the assessment, I applied SQL best practices to write clear, efficient, and well-formatted queries while gaining practical experience in querying relational databases. This project showcases my understanding of MySQL syntax, database concepts, and problem-solving skills commonly required in data analytics and database management roles.

**Key Topics Covered:**

* Database setup using MySQL Workbench
* SQL Theory and Database Fundamentals
* SELECT, WHERE, ORDER BY, LIMIT
* Filtering with IN, BETWEEN, and LIKE
* Aggregate Functions (COUNT, SUM, AVG, MIN, MAX)
* GROUP BY and HAVING
* CASE Statements
* NULL Handling with IS NULL, IFNULL, and COALESCE
* Sorting and Data Analysis
* Writing optimized and readable SQL queries

This assessment reflects my practical understanding of SQL fundamentals and demonstrates my ability to analyze, retrieve, and manipulate data using relational database techniques.
